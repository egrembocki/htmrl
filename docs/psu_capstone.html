 94

 96        :param input_value: The value we want encoded.
 97        :type input_value: float
 98        :return: Returns a list of integers that signify an SDR.
 99        :rtype: list[int]

106
107        index = int(input_value / self._resolution)
108
109        for offset in range(self._active_bits):
110            hash_buffer = index + offset

112            """
113                The lower case i in the struct.pack makes this take in signed 32 bit integers.
114                It is important to note the previous iteration used an upper case I which
115                made this not take negative values. The struct.pack converts an integer
116                into a byte representation.

122                properties of the encoder in unit tests and quantify how significant
123                the hash collisions are.  This encoder can not fix the collisions
124                because it does not record past encodings.  Collisions cause small
125                deviations in the sparsity or semantic similarity, depending on how
